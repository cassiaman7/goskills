# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [Unreleased]

### 新增特性
- **国际化 (Localization)**: 全面支持中文界面和日志。
  - `agent-web` 用户界面汉化。
  - `agent` 核心包及子Agent 的日志和提示词汉化。
- **PPT 图片集成**: PPT 子Agent 现在可以从源内容中提取图片并用于幻灯片生成。
- **UI 交互优化**:
  - 任务执行期间禁用输入，防止重复提交。
  - 修复了任务状态图标（旋转/完成/失败）在中文日志下的显示问题。
  - 优化了 PPT 和播客生成选项的交互体验。
- **PPT 生成**: 支持使用 Slidev 生成 PowerPoint 风格的演示文稿 (#7)。
  - 实现 `PPTSubagent` 生成 Markdown 并构建幻灯片。
  - 添加酷炫视觉效果：玻璃拟态、渐变、动画。
  - 集成 Web 界面以查看生成的 PPT。
- **Web 界面**: 引入基于 Web 的聊天界面 (#9)。
  - 添加 `agent-web` 命令。
  - 实现实时事件流 (SSE)。
  - 支持直接在浏览器中查看报告、播客和 PPT。
- **播客生成**: 添加 `PodcastSubagent` 从报告生成音频脚本 (#6)。
- **报告图片**: 支持在生成的报告中包含图片 (#11)。
- **MCP 支持**: 添加对 Model Context Protocol (MCP) 的支持 (#3)。
- **新技能**: 添加 Tavily 搜索工具并集成 `staruhub/ClaudeSkills` 中的技能。
- **Agent 核心**: 实现核心 Agent 逻辑，包括规划、子Agent 执行和上下文管理 (#5)。

### 改进
- **PPT 美学**: 增强 PPT 视觉风格，采用暗色主题、玻璃拟态和动画 (#7)。
- **CLI**: 改进 `agent-cli` 的提示和输出格式 (#9, #5)。
- **上下文管理**: 重构上下文处理，以便在子Agent 之间更好地管理状态。
- **测试**: 更新测试以覆盖新技能和子Agent。

### 修复
- 修复 Web UI 中的播客标签页渲染问题。
- 修复单元测试失败。
- 修复 GitHub Actions 工作流。
- 修复 README 中的 `action badge.svg`。

### 文档
- 更新 README，包含新功能和使用说明。
- 添加 `agent-cli` 文档。
- 更新网站内容。
